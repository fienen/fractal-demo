Illustrated Bullets

This adds a simple, illustrated “bullet” style embellishment used to set off a section of text similar to an unordered list. This treatment can be applied to any block level element by applying the .illustrated-bullet class to the element.

<div class="illustrated-bullet">
    <!-- Any content can follow in the div -->
    <h4 class="h5 margin-bottom-1">Identify Top Talent</h4>

    <p class="margin-bottom-0">
        Use Aquent’s deep industry-specific talent pool and recruiting<br> connections to identify top candidates.
    </p>
</div>
